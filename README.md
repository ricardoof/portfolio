# Portfolio

Portfólio com seções de sobre, projetos, pesquisa, tecnologias, certificados e contatos.


## Tecnologias

- [Elixir](https://elixir-lang.org/docs.html) : Linguagem de programação funcional
- [Phoenix](https://www.phoenixframework.org/) : Framework web baseado em elixir
- [TailwindCSS](https://tailwindcss.com/) : Estilos
- [Ecto](https://ecto.hexdocs.pm/Ecto.html) : Biblioteca de acesso e persistência de dados (ORM)
